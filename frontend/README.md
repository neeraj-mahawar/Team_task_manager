**TeamTask AI ek modern task management system hai jo Admins aur Members ko projects aur tasks par real-time collaborate karne ki permission deta hai**.

👤 User Roles
Admin: Projects create karna, tasks assign karna, aur puri team ki productivity track karna.

Member: Assigned projects dekhna, task status update karna, aur apni progress track karna.

🛠 Tech Stack
Frontend: React.js, Tailwind CSS, Axios, React Router.

Backend: Node.js, Express.js, MongoDB (Mongoose).

Security: JWT (JSON Web Tokens) for auth, BcryptJS for password hashing.

**Backend Run**
cd backend
nodemon server.js

**Frontend Run**
cd frontend
npm run dev
